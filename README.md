# Bitcrush Mobile

A 2D offline game develop using Python and Kivy (Virtual Environment) to make compatible in to a mobile devices.

### Installation (For Development)
- Clone the repository.

```bash
git clone https://github.com/tambayNgOrtigasAvenue/bitcrush-mobile.git
```

- After cloning, open command prompt and change directory in where you clone the project.

```bash
cd C:\Users\YourName\Projects\bitcrush-mobile
```

- Setup and Install Kivy (Virtual Environment)

1. Setup

```bash
python -m pip install --upgrade pip setuptools virtualenv
```

2. Initialize a env folder

```bash
python -m venv kivy_venv
```

3. Activate

```bash
kivy_venv\Scripts\activate
```

4. Install Kivy

```bash
pip install kivy
```
